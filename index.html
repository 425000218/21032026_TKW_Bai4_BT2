Danh Sách Bài Tập

Chào mừng bạn đến với chuỗi bài tập thiết kế web. Dưới đây là danh sách các bài tập với mô tả chi tiết:

Các Bài Tập

  • 📄 BT1.html - About Us Page (Flexbox Layout)

    Tạo trang About Us với layout Flexbox, card thành viên responsive, và design hiện đại.

  • 📄 BT2.html - Layout & Navigation

    Bài tập cơ bản với header, content, và layout đơn giản.

  • 📄 BT3.html - Blog Layout (2-Column Design)

    Tạo layout 2 cột với sidebar và main content, navbar với Flexbox.

  • 📄 BT4.html - Responsive Website Design

    Giao diện responsive chuẩn với header xanh teal, sidebar 30% - main content 70%, CSS Variables.

Nội Dung Học Tập

1. Cấu Trúc HTML Cơ Bản

Tất cả bài tập đều sử dụng cấu trúc HTML5 semantic với các phần:

  • Header - Phần tiêu đề trang
  • Navigation Bar - Thanh điều hướng
  • Content - Nội dung chính
  • Footer - Chân trang

2. CSS Flexbox

Sử dụng Flexbox để tạo layout responsive mà không cần Bootstrap hoặc Grid.

3. Responsive Design

Trang web tự động điều chỉnh kích thước phù hợp với màn hình desktop, tablet và mobile.
